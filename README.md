

Introduction:
===================
MacroCoin - based of Litecoin used scrypt as a proof of work scheme.
 - 2 mins block targets
 - subsidy halves in 50k blocks
 - 200,000 total coins
 - 0.75 coins per block
 - retarget difficulty every day
